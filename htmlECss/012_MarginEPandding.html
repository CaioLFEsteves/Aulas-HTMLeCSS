Todo elemento do HTML tem Conteúdo, Borda, Padding e Margin


PADDING: Espaçamento interno do elemento, distaciamento entre o conteúdo e a borda, em todas as direções.
MARGIN: Espaçamento externo do elemento, além da borda em todas a direções. Elas não levam em conta a
margem do elemento acima ou abaixo para gerar o afastamento em tela, elas começam a contar o Espaçamento
a bartir da borda do elemento. Isso quer dizer que as margens não se reconhecem, e se cruzam uma sobre a outra.

Vale lembrar que as tags, quaisquer que sejam, por padrão, tem uma margin de tamanho específico.
Então, caso queira da para zerar pelo css, e fazer dois elementos se encostem, como mostrado abaixo.
Isso também se aplica ao body. Para encostar nos cantos é necessário zerá-lo, como no texto do topo.

Para selecionar objetos específicos pode-se utilizar "id=tituloQualquer" e para chamar o id no css, basta usar a #. Pode ser usado apenas para um elemento.

Além do id, também é possível usar o "class=TituloQualquer", mas para chamar no CSS, usa-se ' . ' . Pode ser usado para diversos elementos.



E claro, o mais básico échamar diretamente pelo nome do elemento. O problema éque se usar essa forma, todos os elementos iguais serão afetador de uma vez.
