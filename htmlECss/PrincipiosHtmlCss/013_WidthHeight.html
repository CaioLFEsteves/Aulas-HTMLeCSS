Caracteristicas de width e height (largura e altura), permite acrescentar
algumas formas de medida, como quantidade de pixel, porcentagem, preenchimento automatico,
e varios outros. A primeira é literal, as outras são em relação ao espaço disponível em tela.

Vale lembrar que se usar largura automática para imagem, esta ficará em seu tamanho natural.
Ou seja, sempre é melhor definir a quantidade de pixels.

Para selecionar objetos específicos pode-se utilizar "id=tituloQualquer" e para chamar o id no css, basta usar a #. Pode ser usado apenas para um elemento.

Ao definir uma largura na tag que contêm a imagem filha, e definir a largura da imagem filha como inherit, está irá herdar a largura definida na tag Pai.

Mesmo que definindo o width como automático, ao se definir o limitador de largura mínima, está obedecerá, e ficará com 600px, nesse caso de exemplo, mesmo que naturalmente seja menor.

Mesmo erdando uma largura maior, ao se definir a largura máxima, essa ordem será obedecida, e imagem ficará com no máximo 500px, nesse caso de exemplo



E claro, o mais básico échamar diretamente pelo nome do elemento. O problema éque se usar essa forma, todos os elementos iguais serão afetador de uma vez.
